# 예산 관리 지침

## 1. 예산 편성
- 매년 11월 차년도 예산안 작성
- 부서별 예산 요청서 취합 (10월 말까지)
- 대표이사 승인 후 확정

## 2. 예산 집행
### 2.1 지출 승인 기준
| 금액 | 승인 권한 |
|------|-----------|
| 10만원 미만 | 팀장 승인 |
| 10~100만원 | 부서장 승인 |
| 100~500만원 | 이사 승인 |
| 500만원 이상 | 대표이사 승인 |

### 2.2 지출 절차
1. 지출품의서 작성
2. 승인권자 결재
3. 집행 및 증빙 수집
4. 회계 전표 처리

## 3. 경비 정산
- 법인카드 사용 원칙
- 개인 경비 사용 시 7일 이내 정산
- 영수증 필수 첨부

## 4. 월간 보고
- 매월 5일까지 전월 실적 보고
- 예산 대비 집행률 분석
- 이상 항목 별도 보고
