# 개발 환경 설정 가이드

## 1. 필수 소프트웨어
- **IDE**: VS Code (권장) 또는 IntelliJ
- **Git**: 최신 버전
- **Node.js**: v20 LTS
- **Python**: 3.11 이상
- **Docker**: Desktop 최신 버전

## 2. 저장소 접근
1. GitHub 계정 생성 및 조직 초대 수락
2. SSH 키 등록
3. 저장소 클론:
   ```bash
   git clone git@github.com:hutech-c/metatrans.git
   ```

## 3. 로컬 개발 환경
```bash
# 백엔드
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

# 프론트엔드
cd frontend
npm install
npm run dev
```

## 4. 브랜치 전략
- `main`: 프로덕션
- `develop`: 개발 통합
- `feature/*`: 기능 개발
- `hotfix/*`: 긴급 수정

## 5. 코드 리뷰
- PR 생성 시 최소 1명 리뷰어 지정
- CI 통과 필수
- Squash merge 사용
