# 메타트랜스 TTS(Text-to-Speech) 사용 가이드

## 1. 개요
메타트랜스의 TTS 기능은 텍스트를 자연스러운 음성으로 변환하는 서비스입니다.

## 2. 지원 사항
- 출력 형식: MP3, WAV
- 지원 언어: 한국어, 영어, 일본어, 중국어 등 25개 언어
- 음성 종류: 언어별 2~5종 (남성/여성)

## 3. 사용 방법

### 3.1 텍스트 입력
1. 메타트랜스 로그인
2. [TTS] 메뉴 클릭
3. 텍스트 직접 입력 또는 파일 업로드 (.txt, .docx)

### 3.2 음성 설정
- **언어 및 화자**: 원하는 언어와 음성 선택
- **속도**: 0.5x ~ 2.0x 조절
- **톤**: 높낮이 조절 가능

### 3.3 변환 및 다운로드
1. [미리듣기]로 확인
2. [변환] 클릭
3. MP3/WAV 형식으로 다운로드

## 4. 활용 사례
- 번역 결과물의 음성 검수
- 다국어 프레젠테이션 음성 제작
- 교육 자료 음성 변환
