# 메타트랜스 SST(Speech-to-Text) 사용 가이드

## 1. 개요
메타트랜스의 SST 기능은 음성 파일을 텍스트로 변환하는 서비스입니다.

## 2. 지원 형식
- 음성: MP3, WAV, M4A, FLAC
- 최대 파일 크기: 500MB
- 지원 언어: 한국어, 영어, 일본어, 중국어 등 30개 언어

## 3. 사용 방법

### 3.1 파일 업로드
1. 메타트랜스 로그인
2. [SST] 메뉴 클릭
3. [파일 업로드] 또는 드래그 앤 드롭

### 3.2 설정
- **언어 선택**: 원본 음성의 언어 선택
- **화자 분리**: 다중 화자 구분 여부
- **타임스탬프**: 시간 표시 여부

### 3.3 변환 실행
1. [변환 시작] 클릭
2. 진행률 확인 (평균 음성 길이의 1/3 소요)
3. 완료 후 텍스트 편집 및 다운로드

## 4. 팁
- 배경 소음이 적은 음성일수록 정확도 향상
- 전문 용어가 많은 경우 용어집 등록 활용
