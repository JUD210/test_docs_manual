# 전문가 매칭 절차

## 1. 개요
통번역 프로젝트에 적합한 전문가를 매칭하는 절차입니다.

## 2. 매칭 프로세스

### 2.1 의뢰 등록
1. 프로젝트 정보 입력 (언어쌍, 분야, 분량, 납기)
2. 요구 자격 설정 (AITe 등급, 경력, 전문 분야)
3. 예산 범위 설정

### 2.2 후보자 검색
- 시스템 자동 매칭 (적합도 점수 기반)
- 수동 검색 필터: 언어, 분야, 지역, 등급

### 2.3 매칭 확정
1. 후보자 프로필 검토
2. 샘플 번역 요청 (선택)
3. 계약 조건 협의
4. 매칭 확정 및 프로젝트 시작

## 3. 전문가 등급
| 등급 | 조건 |
|------|------|
| S급 | AITe 고급 + 경력 10년 이상 |
| A급 | AITe 중급 + 경력 5년 이상 |
| B급 | AITe 기초 + 경력 2년 이상 |
| C급 | AITe 기초 합격 |
