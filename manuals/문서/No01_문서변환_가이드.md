# 문서 변환 가이드

## 1. 개요
메타트랜스의 문서 변환 기능은 다양한 문서 형식 간 변환을 지원합니다.

## 2. 지원 변환

| 입력 형식 | 출력 형식 |
|-----------|-----------|
| PDF | DOCX, TXT, XLSX |
| DOCX | PDF, TXT, HTML |
| PPTX | PDF, DOCX |
| XLSX | CSV, PDF |
| 이미지(JPG/PNG) | TXT (OCR) |

## 3. 사용 방법
1. [문서 변환] 메뉴 접속
2. 원본 파일 업로드
3. 출력 형식 선택
4. [변환] 클릭
5. 결과 파일 다운로드

## 4. OCR 기능
- 이미지 내 텍스트 자동 인식
- 한/영/일/중 지원
- 표 구조 인식 가능
- 손글씨는 정확도 낮음 (70% 이하)

## 5. 주의사항
- 암호화된 PDF는 변환 불가
- 최대 파일 크기: 100MB
- 복잡한 레이아웃은 수동 보정 필요
